# -Network-Port-Scanner-Simulator
A Python-based network security project that simulates port scanning and service identification.
## Features
- Detects open ports
- Identifies common services
- Flags potentially risky ports
- Generates a security assessment report

## Technologies Used
- Python
- Network Security Concepts
- Security Monitoring

## Sample Findings
- SSH (22)
- HTTP (80)
- HTTPS (443)
- RDP (3389)
